Extends WooCommerce stock options by enabling set minimum/maximum product quantity per order

== Description ==

Adds option to set minimum/maximum product quantity in an order on product level and/or category level.

= Features: =

* Option to set minimum product quantity in an order;
* Option to set maximum product quantity in am order;
* Option to recursively use category level settings (product will inherit first non-empty value from category hierarcy);
* Plugin adds "Select quantity" input field to each product.

= Supported languages =

* English

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/adk_minmax_product` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Proceed to Admin > WooCommerce > Products > Product > Stock and set minimum/maximun product qiantity per order.
1. Or proceed to Admin > WooCommerce > Products > Categories and set this option for entire category.
